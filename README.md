# 영화 추천 챗봇 🎬

터미널 기반의 영화 추천 챗봇입니다. 고객에게 객관식 질문을 통해 선호도를 파악하고 맞춤형 영화를 추천해드립니다.

## 기능

- 객관식 질문으로 사용자의 취향 파악
- 질문에 대한 답변 기반 맞춤형 영화 추천
- 언제든지 '0' 입력으로 프로그램 종료 가능

## 사용 방법

```bash
python movie_recommender.py
```

## 질문 내용

1. **선호하는 장르**: 액션, 코미디, 드라마, 공포, 로맨스, SF 중 선택
2. **선호하는 분위기/스타일**: 장르에 따라 다양한 선택지 제공
3. **시청 시간대 선호도**: 저녁, 주말 오후, 밤 늦게 중 선택

## 종료 방법

- 질문 중 언제든지 `0`을 입력하면 프로그램이 종료됩니다.
- `Ctrl+C`로도 중단할 수 있습니다.

## 요구사항

- Python 3.6 이상
